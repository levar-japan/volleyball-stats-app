# 統計分析 - 統計内容一覧

## 全体統計

チーム全体の総合的な統計情報を表示します。

### 基本統計
- **試合数**: 記録された試合の総数
- **総得点**: 全試合での総得点数（アタック、サーブ、ブロックの合計）
- **総失点**: 全試合での総失点数
- **セット勝率**: セットの勝率（勝ったセット数 / 総セット数 × 100）

### スキル別統計
各スキルの成功率と詳細を表示：

1. **アタック**
   - 成功率（%）
   - 得点数 / 総試行回数
   - 失点数

2. **サーブ**
   - 成功率（%）
   - 得点数 / 総試行回数
   - 失点数

3. **ブロック**
   - 成功率（%）
   - 得点数 / 総試行回数
   - 失点数

4. **レセプション**
   - 成功率（%）（Aパス + Bパス / 総試行回数）
   - A+Bパス数 / 総試行回数
   - 失点数

5. **ディグ**
   - 成功率（%）
   - 成功数 / 総試行回数
   - 失敗数

### 平均値
- **試合平均得点**: 1試合あたりの平均得点数
- **試合平均失点**: 1試合あたりの平均失点数

### グラフ表示
- **チームパフォーマンス推移**: 試合ごとの総得点と総失点の推移
- **成功率推移**: 試合ごとのアタック、サーブ、レセプション成功率の推移
- **弱点分析**: 失点が多い順に表示（バーグラフ）

## 選手別統計

各選手の個別統計情報を表示します。

### 選手一覧表示
全選手の統計を一覧で表示：
- 選手名
- 出場試合数
- 総得点
- 総失点
- 各スキル（アタック、サーブ、ブロック、レセプション）の成功率

### 個別選手統計
選択した選手の詳細統計：

#### 基本統計
- **出場試合数**: その選手が参加した試合数
- **総得点**: その選手の総得点数
- **総失点**: その選手の総失点数
- **得失点差**: 総得点 - 総失点

#### スキル別統計
各スキルの成功率と詳細：
- **アタック**: 成功率、得点数/試行回数、失点数
- **サーブ**: 成功率、得点数/試行回数、失点数
- **ブロック**: 成功率、得点数/試行回数、失点数
- **レセプション**: 成功率、A+Bパス数/試行回数、失点数
- **ディグ**: 成功率、成功数/試行回数、失敗数

#### パフォーマンス推移
試合ごとのパフォーマンス推移をグラフで表示：
- アタック得点推移
- サーブ得点推移
- ブロック得点推移

## 試合別統計

各試合の詳細統計情報を表示します。

### 試合一覧表示
全試合の統計を一覧で表示：
- 対戦相手
- 試合日
- セット結果（勝敗数と総スコア）
- 総得点
- 総失点
- 各スキル（アタック、サーブ、レセプション）の成功率

### 個別試合統計
選択した試合の詳細統計：

#### 試合基本情報
- **対戦相手**: 相手チーム名
- **試合日**: 試合が行われた日付
- **セット結果**: 勝ったセット数と負けたセット数、総スコア
- **試合結果**: 勝利/敗北/引き分け

#### セットスコア
各セットのスコアを表示：
- Set 1, Set 2, Set 3... のスコア
- 勝ったセットは緑色、負けたセットは赤色で表示

#### スキル別統計
その試合での各スキルの成功率と詳細：
- **アタック**: 成功率、得点数/試行回数、失点数
- **サーブ**: 成功率、得点数/試行回数、失点数
- **ブロック**: 成功率、得点数/試行回数、失点数
- **レセプション**: 成功率、A+Bパス数/試行回数、失点数
- **ディグ**: 成功率、成功数/試行回数、失敗数

#### 得点・失点サマリー
- **総得点**: その試合での総得点数
- **総失点**: その試合での総失点数

## 統計の計算方法

### 成功率の計算
- **アタック成功率** = (得点数 / 総試行回数) × 100
- **サーブ成功率** = (得点数 / 総試行回数) × 100
- **ブロック成功率** = (得点数 / 総試行回数) × 100
- **レセプション成功率** = ((Aパス数 + Bパス数) / 総試行回数) × 100
- **ディグ成功率** = (成功数 / 総試行回数) × 100

### 得点の計算
- **総得点** = アタック得点 + サーブ得点 + ブロック得点

### 失点の計算
- **総失点** = アタック失点 + サーブ失点 + ブロック失点 + レセプション失点 + ディグ失敗 + 自チームエラー

### セット勝率の計算
- **セット勝率** = (勝ったセット数 / 総セット数) × 100

## フィルター機能

- **シーズン選択**: 特定のシーズンのデータのみを表示
- **選手選択**: 特定の選手のデータのみを表示（選手別統計）
- **試合選択**: 特定の試合のデータのみを表示（試合別統計）
